# 《西游记》书评

**作者：吴承恩**
**出版社：岳麓书社**
**出版时间：2004年6月**
**页数：688页**
**豆瓣评分：9.4**
**评论者：Aniseedaliao**
**日期：2026年4月18日**

---

## 一、四大名著中的"另类"

在中国四大名著中，《西游记》是最"另类"的一部。

《红楼梦》是世情小说的巅峰，写尽人间悲欢。
《三国演义》是历史演义的典范，演绎权谋争霸。
《水浒传》是英雄传奇，讲述草莽豪杰。

而《西游记》是什么？

- 是神魔小说？是的，充满了神仙妖怪。
- 是冒险故事？是的，九九八十一难。
- 是寓言哲学？是的，修行悟道的隐喻。
- 是讽刺文学？是的，影射人间百态。

**《西游记》是所有这些，又超越了所有这些。**

豆瓣9.4分，超过77%的读者给予五星评价。这不仅是一部神话小说，更是中国人心中的"精神原型"——那个敢于大闹天宫的猴子，已经成为反抗与自由的象征。

## 二、历史与传说：从玄奘取经到《西游记》

《西游记》的故事，源于真实的历史事件。

### 2.1 玄奘取经（627-645年）

**唐僧的原型——玄奘法师：**

| 事实 | 内容 |
|------|------|
| 时间 | 唐太宗贞观元年（627年）出发 |
| 年龄 | 25岁 |
| 路线 | 长安→中亚→阿富汗→巴基斯坦→印度 |
| 历时 | 前后19年 |
| 成果 | 带回佛经657部 |

玄奘的西行是一次传奇式的万里长征，轰动一时。他口述的《大唐西域记》记录了沿途各国的历史、地理。

### 2.2 从历史到神话

真实的历史如何变成神话？

| 阶段 | 作品 | 特点 |
|------|------|------|
| 唐代 | 《大唐西域记》 | 史实记录 |
| 唐代 | 《大唐大慈恩寺三藏法师传》 | 增添神话色彩 |
| 南宋 | 《大唐三藏取经诗话》 | 故事雏形 |
| 金代 | 院本《唐三藏》《蟠桃会》 | 戏曲化 |
| 元代 | 杂剧《唐三藏西天取经》 | 戏剧形式 |
| 明代 | 《西游记》吴承恩 | 集大成 |

吴承恩正是在民间传说、话本、戏曲的基础上，经过艰苦的再创造，完成了这部伟大的文学巨著。

## 三、作者：科场失意的天才

**吴承恩（约1500-1582）**，字汝忠，号射阳山人，淮安府山阳县（今江苏淮安）人。

### 3.1 家世背景

| 世代 | 身份 |
|------|------|
| 曾祖父、祖父 | 读书人，县学训导、教谕 |
| 父亲吴锐 | 家贫，出赘徐家，成为小商人 |

吴家虽有读书传统，但已沦落为小商人阶层。这种社会地位的变化，对吴承恩产生了深刻影响。

### 3.2 坎坷的人生

**少年得志：**

> "性敏而多慧，博极群书，为诗文，下笔立成。"——天启《淮安府志》

吴承恩从小聪明，早入学，名满乡里。

**中年坎坷：**

科举屡遭挫折，四十多岁才补了一个岁贡生，五十多岁任浙江长兴县丞，后又担任荆王府纪善——都是低级闲职。

**晚年的创作：**

吴承恩创作《西游记》大约在中年以后，或认为是晚年所作。他用一支笔，创造了一个神奇的世界，在这个世界里，弱者可以战胜强者，反抗者可以挑战权威。

**这难道不是他对自己命运的想象性补偿吗？**

## 四、结构：三段式叙事

《西游记》全书一百回，结构清晰，可分为三大部分：

### 4.1 第一部分：大闹天宫（第1-8回）

这是全书最精彩的章节，孙悟空出世、拜师、大闹天宫。

**故事梗概：**

| 情节 | 内容 |
|------|------|
| 石猴出世 | 从石头里蹦出来，天生异禀 |
| 拜师学艺 | 拜菩提祖师为师，学得七十二变、筋斗云 |
| 龙宫借宝 | 取得如意金箍棒 |
| 大闹地府 | 勾掉生死簿上的名字 |
| 大闹天宫 | 自封"齐天大圣"，与天庭对抗 |
| 五行山下 | 被如来佛祖压在五行山下 |

**大闹天宫的意义：**

这是中国文学史上最壮丽的反叛场景之一。一个"卑贱"的石猴，敢于挑战至高无上的玉皇大帝，喊出"皇帝轮流做，明年到我家"的口号。

### 4.2 第二部分：取经缘起（第8-12回）

交代唐僧的出身及取经的缘由。

**故事梗概：**

| 情节 | 内容 |
|------|------|
| 如来赐经 | 如来佛祖有三藏真经，可超度亡魂 |
| 观音寻僧 | 观音菩萨东土寻找取经人 |
| 唐僧出世 | 江流儿的故事 |
| 太宗游地府 | 唐太宗梦游地府，决定设坛超度 |
| 玄奘西行 | 唐僧领旨西行取经 |

这一部分为取经故事提供了宗教和政治的合法性。

### 4.3 第三部分：西天取经（第13-100回）

这是全书的主体，唐僧师徒历经九九八十一难，终于取到真经。

**师徒四人：**

| 人物 | 前身 | 法号 | 兵器 | 性格 |
|------|------|------|------|------|
| 唐僧 | 金蝉子 | 三藏 | 无 | 慈悲、懦弱 |
| 孙悟空 | 石猴 | 行者 | 金箍棒 | 勇敢、机智 |
| 猪八戒 | 天蓬元帅 | 悟能 | 九齿钉耙 | 憨厚、贪吃 |
| 沙僧 | 卷帘大将 | 悟净 | 降妖杖 | 忠诚、沉默 |

**八十一难：**

从"金蝉遭贬"到"第九九八十一难"（通天河老鼋湿经书），每一难都是一个独立的故事，又环环相扣。

## 五、人物：四种人格的象征

《西游记》的人物不只是"角色"，更是人格的象征。

### 5.1 孙悟空：英雄与心猿

孙悟空是《西游记》的第一主人公，是中国文学史上最光彩夺目的英雄形象之一。

**能力：**

| 神通 | 说明 |
|------|------|
| 七十二变 | 随心所欲的变化 |
| 筋斗云 | 一个筋斗十万八千里 |
| 火眼金睛 | 识破一切妖魔伪装 |
| 如意金箍棒 | 重一万三千五百斤，随心变化 |

**性格特征：**

- 勇敢：天不怕地不怕
- 机智：善于动脑筋
- 叛逆：敢于挑战权威
- 忠诚：对师父不离不弃
- 诙谐：爱开玩笑

**象征意义：**

孙悟空是"心猿"——人心的象征。人心躁动不安，如猿猴跳跃；但经过修炼，可以成佛。《西游记》的取经之路，实际上是一个"修心"的过程。

### 5.2 猪八戒：欲望与人性

猪八戒是一个喜剧形象，也是最受欢迎的角色之一。

**前身：**

天蓬元帅，因调戏嫦娥被贬下凡，错投猪胎。

**性格特征：**

- 憨厚：老实本分
- 贪吃：好吃懒做
- 贪财：藏私房钱
- 好色：念念不忘高老庄的媳妇
- 怕难：常打退堂鼓
- 挑拨：爱说孙悟空坏话

**象征意义：**

猪八戒是"欲望"的象征——食欲、色欲、财欲、惰性。这些欲望是人的本能，难以根除。但猪八戒不是反面人物，他憨厚可爱，是取经队伍中不可或缺的一员。

**吴承恩对猪八戒的态度：**

> **作者对猪八戒缺点的批评是严厉的，但又是善意的。他并不是一个被否定的人物，因此人们并不厌恶猪八戒，相反却感到十分真实可爱。**

### 5.3 唐僧：信仰与软弱

唐僧是取经的核心人物，但形象相对单薄。

**特点：**

- 慈悲：心地善良
- 虔诚：一心取经
- 软弱：没有本领，常被妖怪抓走
- 固执：不听孙悟空劝告，常念紧箍咒

**争议：**

很多读者不喜欢唐僧，认为他无能、固执、是非不分。但也有观点认为，唐僧代表的是"信仰"——没有武力，没有神通，但有一颗坚定的心。

### 5.4 沙僧：忠诚与沉默

沙僧是最不起眼的角色，形象模糊。

**特点：**

- 忠诚：任劳任怨
- 沉默：很少说话
- 调和：常在师父和师兄之间调停

**象征意义：**

沙僧代表的是"戒"——持戒、安分、守己。他没有孙悟空的张扬，没有猪八戒的欲望，是最稳定的因素。

### 5.5 四人一体的隐喻

取经团队四人，其实是"一个人"的不同侧面：

| 人物 | 象征 |
|------|------|
| 唐僧 | 信仰（灵魂） |
| 孙悟空 | 心（心智） |
| 猪八戒 | 欲望（本能） |
| 沙僧 | 戒律（意志） |

取经的过程，就是信仰引导心智，约束欲望，持守戒律，最终达到觉悟的过程。

## 六、主题：多重解读的可能

《西游记》的魅力在于它可以被多重解读。

### 6.1 修行寓言

最直接的解读：**《西游记》是一部修行成佛的寓言。**

**证据：**

- 孙悟空是"心猿"，需要被收服
- 猪八戒代表"欲望"，需要被节制
- 九九八十一难，是修行的考验
- 取得真经，是觉悟的完成

**心经的反复出现：**

《西游记》多次引用《般若波罗蜜多心经》，强调"色即是空，空即是色"的道理。取经之路，是"渡过苦海"的修行。

### 6.2 社会讽刺

《西游记》也是一部讽刺小说，影射明代社会的黑暗。

**讽刺对象：**

| 对象 | 讽刺内容 |
|------|----------|
| 玉皇大帝 | 昏聩无能的统治者 |
| 天庭官僚 | 推诿扯皮的官员 |
| 妖怪 | 有后台的恶势力（常是天庭神仙的坐骑、童子） |
| 人间国王 | 昏君、暴君 |

**"有后台"的妖怪：**

一个有趣的现象：很多妖怪都是天庭神仙的坐骑或童子下凡。孙悟空要打死它们，总有神仙来"救走"。而没有后台的妖怪，往往被一棒打死。

这是对现实社会的辛辣讽刺——**有后台的人作恶可以逍遥法外，没后台的人只能被当作替罪羊。**

### 6.3 反抗精神

孙悟空大闹天宫，是中国文学史上反抗精神的巅峰表达。

**名句：**

> **"皇帝轮流做，明年到我家！"**

这句话在封建社会是惊世骇俗的。它否定了"君权神授"的合法性，主张权力可以轮替，皇帝可以改变。

**吴承恩的隐秘心声：**

作为一个科举失意、终身沉沦下僚的知识分子，吴承恩通过孙悟空的形象，表达了对社会不公的愤怒，对等级制度的抗议。

### 6.4 冒险与成长

《西游记》也是一部成长小说。

**孙悟空的成长：**

| 阶段 | 状态 |
|------|------|
| 石猴 | 天真无邪 |
| 美猴王 | 自由自在 |
| 齐天大圣 | 狂妄反叛 |
| 孙行者 | 收敛磨练 |
| 斗战胜佛 | 成就正果 |

从石猴到佛，是成长的历程——从天真到叛逆，从叛逆到磨砺，从磨砺到觉悟。

## 七、艺术特色：神魔世界的创造

### 7.1 想象力

《西游记》的想象力是中国文学的巅峰。

**创造的神话世界：**

- 花果山：猴子的乌托邦
- 天庭：等级森严的官僚机构
- 地府：阴森恐怖的审判场所
- 龙宫：海底的宝库
- 西天：极乐世界

每一个空间都有完整的设定和逻辑。

**神奇的法术：**

- 七十二变
- 筋斗云
- 火眼金睛
- 定身法
- 分身术

这些法术后来成为中国文化中"神通"的典范。

### 7.2 人物塑造

孙悟空和猪八戒是中国文学史上最成功的艺术形象之一。

**孙悟空：**

- 有大英雄的不凡气度
- 有爱听恭维话的缺点
- 机智勇敢又诙谐好闹
- 敢斗——与天斗、与地斗、与妖魔斗

**猪八戒：**

- 憨厚老实，有力气
- 满身毛病——贪吃、贪财、好色、怕难
- 但不是被否定的人物
- 真实可爱，有血有肉

### 7.3 语言艺术

《西游记》的语言生动活泼，充满幽默感。

**诙谐的风格：**

孙悟空的对话总是充满智慧和幽默。他骂妖怪、嘲弄神仙、调侃师父，语言生动有趣。

**诗词的运用：**

每回开头和结尾常有诗词，描写景物、概括情节，增加了文学性。

## 八、争议：大闹天宫前后判若两人？

一个常见的批评：**孙悟空大闹天宫时战无不胜，取经路上却常打败仗，要请神仙帮忙。这是不是前后矛盾？**

### 8.1 常见解释

**（1）放水说：**

孙悟空被压五行山五百年，本领没有退化，但他学会了"藏拙"。很多妖怪有后台，他不想得罪太多人。

**（2）束缚说：**

紧箍咒限制了孙悟空的发挥。唐僧的牵制使他不能全力施展。

**（3）修行说：**

取经是修心的过程，不是比武的竞赛。孙悟空需要学会的不是"打赢"，而是"修心"。

### 8.2 作者意图

吴承恩可能有意这样安排：

- 大闹天宫表现反抗精神
- 取经路上表现成长和磨砺
- 战胜妖怪不是目的，取得真经才是目的

## 九、当代意义：为什么今天还要读《西游记》？

### 9.1 反抗精神的象征

孙悟空大闹天宫的形象，已经成为反抗与自由的象征。

- 在任何时代，敢于挑战不公的精神都值得尊敬
- "皇帝轮流做"的思想，是对专制制度的否定
- 弱者可以战胜强者，这是人民的愿望

### 9.2 成长的启示

《西游记》是一部关于成长的书。

- 每个人都有"心猿"——躁动、不安分
- 每个人都有"八戒"——欲望、弱点
- 成长就是约束心猿、节制欲望、坚定信仰

### 9.3 团队的智慧

取经团队四人，各有长短：

- 唐僧有信仰但无能
- 孙悟空有能力但冲动
- 猪八戒憨厚但懒惰
- 沙僧忠诚但平庸

但他们组成了成功的团队——**互补、协作、共同前进。**

### 9.4 民族文化的符号

《西游记》已经成为中国文化的符号：

- 孙悟空是"中国超人"
- "西游记"是冒险的代名词
- 八十一难是考验的象征

## 十、评分与推荐

**评分：★★★★★**

**优点：**
- 想象力丰富，创造完整的神话世界
- 人物生动，孙悟空、猪八戒深入人心
- 语言幽默，诙谐有趣
- 主题丰富，可多重解读
- 既是神话，又是寓言

**阅读提示：**
- 前八回最精彩，建议重点阅读
- 中间八十一难可选择性阅读
- 注意妖怪"有后台"的讽刺意义
- 可以关注修行的隐喻

**推荐给谁：**

- 所有关心中国文化的人
- 想要理解"反抗精神"的人
- 正在经历成长困惑的人
- 喜欢神话与冒险的读者

**阅读建议：**

- 选择有注释的版本
- 可以配合影视作品（86版电视剧最经典）
- 关注孙悟空的成长历程
- 思考：取经的意义是什么？

## 十一、结语：心猿归正，大道得成

《西游记》的开篇写道：

> **"混沌未分天地乱，茫茫渺渺无人见。自从盘古破鸿蒙，开辟从兹清浊辨。"**

从混沌到开天辟地，从无到有，这是一部创世史诗。

而它的结尾写道：

> **"五圣成真，径回东土。"**

孙悟空成为"斗战胜佛"，唐僧成为"旃檀功德佛"，猪八戒成为"净坛使者"，沙僧成为"金身罗汉"，白龙马成为"八部天龙"。

**从石猴到佛，从凡人到圣，这是成长的道路。**

《西游记》告诉我们：

> **人生是一场取经之旅。**
> **路途遥远，困难重重。**
> **但有信仰、有伙伴、有勇气，终能到达彼岸。**

那个大闹天宫的猴子，曾经狂妄地说"皇帝轮流做"；后来他学会了收敛，学会了合作，学会了坚持。他不再是那个只知打打杀杀的"齐天大圣"，而是护送师父西天取经的"孙行者"。

这是成长，不是堕落。

**愿每个人心中都有一只孙悟空——既能大闹天宫，也能取得真经。**
